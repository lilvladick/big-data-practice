Проклятие размерности — явление, при котором с ростом числа признаков данные становятся разреженными, расстояния между точками теряют информативность, а модели требуют экспоненциально больше данных для обучения, что приводит к переобучению, ухудшению обобщающей способности и росту вычислительных затрат. Снижение размерности решает эту проблему двумя путями: Feature Selection — отбор признаков, то есть оценка важности признаков с помощью алгоритмов машинного обучения и отсечение ненужных, и Feature Extraction — извлечение признаков, то есть преобразование данных, специфических для предметной области, в понятные для модели векторы. Регуляризация также применяется как метод снижения размерности: Ridge-регрессия с L2-регуляризацией сжимает веса признаков, а LASSO с L1-регуляризацией обнуляет неважные признаки, автоматически выполняя отбор. Метод главных компонент PCA — линейный метод извлечения признаков, который находит ортогональные направления максимальной дисперсии в данных. Первая главная компонента объясняет наибольшую изменчивость данных, каждая следующая компонента объясняет максимальную оставшуюся дисперсию при условии ортогональности предыдущим. Алгоритм PCA включает центрирование данных, вычисление ковариационной матрицы, нахождение собственных векторов и собственных значений, сортировку собственных векторов по убыванию собственных значений и проекцию данных на выбранные главные компоненты. Собственные значения показывают долю объяснённой дисперсии каждой компонентой. PCA применяется для сжатия данных, визуализации многомерных данных в двух или трёх измерениях, удаления шума и мультиколлинеарности, ускорения обучения моделей. t-SNE — нелинейный метод снижения размерности для визуализации, который сохраняет локальные структуры данных: близкие точки в исходном пространстве остаются близкими в проекции. t-SNE использует распределение Стьюдента для моделирования сходства в низкоразмерном пространстве, что позволяет лучше разделять кластеры по сравнению с классическим SNE, использующим гауссово распределение. UMAP — современная альтернатива t-SNE, которая работает быстрее, лучше сохраняет глобальную структуру данных и масштабируется на большие наборы. Выбор метода снижения размерности зависит от задачи: PCA подходит для линейных зависимостей, сжатия данных и предобработки перед другими алгоритмами, t-SNE и UMAP — для визуализации и разведочного анализа нелинейных структур. При применении методов снижения размерности важно масштабировать признаки, так как методы чувствительны к масштабу данных, и интерпретировать результаты с учётом предметной области.
